# WeatherNow
A Simple Weather App for iOS, using [OpenWeather](https://openweathermap.org/) public API.

## Features
* Simple design
* Automatically get forecast based on user's location
* Supports Light and Dark mode
* Available in:
  * English
  * Portuguese (Brazil)
